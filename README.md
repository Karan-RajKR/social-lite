# SocialLite

A modern, mini social media application built with Express.js and SQLite.

![SocialLite Demo](screenshots/demo.png)

## Features

- **User Profiles**: Create accounts, set avatars (auto-generated initials), and view stats.
- **Feed**: Real-time-like feed of posts from all users (or followed users).
- **Interactions**: Like posts and add comments.
- **Follow System**: Follow/Unfollow users to curate your experience.
- **Responsive Design**: Works on desktop and mobile with a premium dark theme.

## Tech Stack

- **Frontend**: Vanilla HTML, CSS (Variables, Flexbox/Grid), JavaScript (Fetch API).
- **Backend**: Node.js, Express.js.
- **Database**: SQLite (Zero-config, file-based).
- **Auth**: Session-based authentication with `bcrypt` for password hashing.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/Karan-RajKR/social-lite.git](https://github.com/Karan-RajKR/social-lite.git)
    cd social-lite
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the server:
    ```bash
    node server.js
    ```

4.  Open your browser and visit:
    ```
    http://localhost:3000
    ```
